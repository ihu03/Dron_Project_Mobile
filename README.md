# Dron_Project

### 실행 명령어 터미널에 [ $env:PORT=8080; node server.js ] 입력
### 로컬환경 웹 주소 : localhost:8080

### 웹사이트 기체 움직임 속도
### config.json => "pollMs": 500(0.5초) 

### 현재 알고리즘
### 데이터 결합: 실시간 ADS-B 데이터와 fakePlanes를 합쳐(combined) 지도·테이블·통계 모두 처리. fakeSet으로 가짜 마커는 STALE 제거 로직에서 제외.
### 통계/경고 기준: 항상 fakePlanes 중 마지막 추가된 기체를 기준점으로 사용.
### 레이더 내 기체 수: 하버사인 거리 ≤ radarRadiusKm 개수 집계.
### 안전 반경 침범: 거리 ≤ safetyRadiusKm면 즉시 경고.
### ETA 계산: ground speed가 있는 기체만 대상, ETA = (거리-안전반경)/속도(m/s)→분. 최소 ETA 추적, ETA 슬라이더 범위 안이면 목록에 표시.
### 경고 메시지/색상: 침범 시 “안전 반경 침범!”, 그렇지 않으면 최소 ETA 예고, 없으면 “위협 없음”.
